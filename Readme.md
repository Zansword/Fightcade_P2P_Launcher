P2P Launcher For fightcade

-KR-

파이트케이드, 파이트케이드2, 파이트케이드2 오버클럭에디션의 P2P 다이렉트 연결툴입니다.

파이트케이드 서버 없이 서로의 IP주소만 알고 있다면 서로 외부 ip주소를 교환하여 네트워크 플레이가 가능합니다.

개발자 : 참철이(chamcham425)

오리지널 툴 개발자 : 짱구(jicdoo)님

gamelist.cfg - 게임리스트입니다. 메모장으로 수정가능합니다.

settings.ini - 설정파일
